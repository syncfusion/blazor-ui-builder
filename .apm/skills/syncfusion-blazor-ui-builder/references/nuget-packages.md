## ⚠️ CRITICAL: Component-Specific Script References

**IMPORTANT:** Some Syncfusion Blazor components require specific script files in addition to the base `syncfusion-blazor.min.js` script. These MUST be added to your Blazor app entry point (wwwroot/index.html for WASM, or Components/App.razor for WebApp).

### Component Scripts

| Component | NuGet Package | Required Script |
|-----------|---------------|------------------|
| **SfPdfViewer2** | Syncfusion.Blazor.SfPdfViewer | `<script src="_content/Syncfusion.Blazor.SfPdfViewer/scripts/syncfusion-blazor-sfpdfviewer.min.js" type="text/javascript"></script>` |
| **SfSmartPdfViewer** | Syncfusion.Blazor.SfSmartPdfViewer | `<script src="_content/Syncfusion.Blazor.SfSmartPdfViewer/scripts/syncfusion-blazor-sfsmartpdfviewer.min.js" type="text/javascript"></script>` |
| **SfDocumentEditorContainer** | Syncfusion.Blazor.WordProcessor | `<script src="_content/Syncfusion.Blazor.WordProcessor/scripts/syncfusion-blazor-documenteditor.min.js" type="text/javascript"></script>` |
| **SfSpreadsheet** | Syncfusion.Blazor.Spreadsheet | `<script src="_content/Syncfusion.Blazor.Spreadsheet/scripts/syncfusion-blazor-spreadsheet.min.js" type="text/javascript"></script>` |

### Script Placement Rules

**For Blazor WebAssembly (WASM):**
- Add component scripts in `wwwroot/index.html`
- Place after the base Syncfusion Blazor Base script:
```html
<script src="_content/Syncfusion.Blazor.Core/scripts/syncfusion-blazor.min.js" type="text/javascript"></script>
<!-- Add component-specific scripts here -->
<script src="_content/Syncfusion.Blazor.SfPdfViewer/scripts/syncfusion-blazor-sfpdfviewer.min.js" type="text/javascript"></script>
```

**For Blazor WebApp (Server/Auto/WebAssembly):**
- Add component scripts in `Components/App.razor`
- Place after the base Syncfusion Blazor Base script:
```razor
<script src="_content/Syncfusion.Blazor.Core/scripts/syncfusion-blazor.min.js" type="text/javascript"></script>
<!-- Add component-specific scripts here -->
<script src="_content/Syncfusion.Blazor.SfPdfViewer/scripts/syncfusion-blazor-sfpdfviewer.min.js" type="text/javascript"></script>
```

### Why These Scripts Are Required

- **PDF Viewer (SfPdfViewer2)**: Handles PDF rendering, annotations, and advanced viewer features
- **Smart PDF Viewer (SfSmartPdfViewer)**: Provides AI-powered PDF features and smart functionality
- **Document Editor (SfDocumentEditorContainer)**: Manages rich text editing, formatting, and document operations
- **Spreadsheet (SfSpreadsheet)**: Handles spreadsheet calculations, formula evaluation, and data operations

**Without these scripts, components will not render or function correctly.**
